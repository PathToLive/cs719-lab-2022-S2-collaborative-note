Question 1: Github stop support authentification using username and password.What should I do to access my repository?

context:

A student tried to clone own repository from the github account. The Git bash command line firstly prompted for username and password. However, after inputting the credential, the git bash showed "fatal error remote: Repository not found." Occasionally, a message also showed "Support for password authentication was removed on August 13, 2021. "

answer:

Github has announced that they have enforced using personal access token as main authentification measures due to security concers. You will need to generate a personal access token on the gituhb website by following the steps in this link
